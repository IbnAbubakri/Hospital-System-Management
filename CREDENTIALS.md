## Access Levels

### Administrator
- Full access to all system features
- Can manage users, departments, and system settings
- Has access to all reports and analytics
- Can configure system-wide settings

### Doctor
- Access to patient records and clinical data
- Can create and manage EMR entries
- Can view lab results and radiology reports
- Department-specific data access
- Limited access to financial reports

### Auxiliary Nurse
- Access to patient registration and triage
- Can manage patient admissions
- Can schedule appointments
- Can view assigned patient information
- No access to financial or sensitive clinical reports

## Notes

- All passwords are case-sensitive
- Email addresses can be used interchangeably with usernames for login
- Default credentials are for demo/testing purposes only
- In production, users would be required to change passwords on first login
